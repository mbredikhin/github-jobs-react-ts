# Jobba 

Client app for Github Jobs API 

## Run Locally

Clone the project

```bash
  git clone https://github.com/mbredikhin/jobba.git
```

Go to the project directory

```bash
  cd jobba
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
